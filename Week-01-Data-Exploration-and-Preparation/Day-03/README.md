# Day 3 – Customer Segmentation using K-Means and DBSCAN

## Objective

The objective of Day 3 was to segment customers into meaningful groups based on their purchasing behavior using clustering techniques. Customer segmentation helps businesses understand different customer profiles and supports targeted marketing, personalized recommendations, and better business decision-making.

---

## Tasks Performed

### 1. Customer Data Preparation

Prepared the customer dataset for clustering by selecting relevant features and ensuring the data was suitable for analysis.

The following tasks were completed:

- Selected customer-related features.
- Removed unnecessary columns.
- Standardized the data for clustering.
- Prepared the dataset for machine learning algorithms.

### 2. K-Means Clustering

Implemented the K-Means clustering algorithm to group customers with similar purchasing behavior.

The following steps were performed:

- Determined the optimal number of clusters.
- Trained the K-Means model.
- Assigned customers to different clusters.
- Visualized the resulting customer groups.

### 3. DBSCAN Clustering

Applied the DBSCAN clustering algorithm to identify clusters based on data density and detect noise or outliers.

The following tasks were completed:

- Configured DBSCAN parameters.
- Identified customer clusters.
- Detected outliers within the dataset.
- Compared clustering results with K-Means.

### 4. Cluster Evaluation

Evaluated the quality of the generated clusters using appropriate evaluation techniques and visual analysis.
