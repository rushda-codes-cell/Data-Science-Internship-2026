# Day 6 – LSTM Model Development using PyTorch Lightning

## Objective

The objective of Day 6 was to develop a Long Short-Term Memory (LSTM) model for demand forecasting using PyTorch Lightning. LSTM networks are designed to learn patterns from sequential data, making them suitable for time series forecasting tasks.

---

## Tasks Performed

### 1. Data Preparation

Prepared the time series dataset for training the LSTM model.

The following tasks were completed:

- Organized the sales data into sequential format.
- Normalized the dataset for better model performance.
- Created input and output sequences.
- Split the dataset into training and testing sets.

### 2. LSTM Model Development

Built an LSTM neural network using PyTorch Lightning.

The following steps were performed:

- Defined the LSTM architecture.
- Configured model parameters.
- Initialized the training process.
- Trained the model using historical sales data.

### 3. Model Evaluation

Evaluated the model's forecasting performance by comparing predicted values with actual sales.

The evaluation included:

- Training loss analysis
- Validation loss analysis
- Forecast comparison
- Performance visualization

### 4. Result Visualization

Visualized the forecasting results to understand how well the model captured sales patterns over time.

---

---

## Outcome

By the end of Day 6, an LSTM-based forecasting model was successfully developed and trained on historical sales data. The model demonstrated the ability to learn temporal patterns from sequential data and generate demand forecasts, providing a strong foundation for advanced forecasting techniques.
