# Week 1 – Data Exploration & Preparation

## Overview

The first week of the internship focused on understanding and preparing the retail sales dataset for machine learning. During this week, I explored the dataset, performed data cleaning and feature engineering, segmented customers, prepared time-series data, and developed baseline forecasting models.

The primary objective was to build a strong data foundation before moving on to advanced machine learning models and deployment in the following weeks.

## Topics Covered

### Day 1
- Dataset Selection
- Exploratory Data Analysis (EDA)
- Distribution Analysis
- Missing Value Analysis
- Correlation Heatmap

### Day 2
- Data Cleaning
- Feature Engineering
- RFM Analysis
- Rolling Statistics
- Data Validation using Great Expectations

### Day 3
- Customer Segmentation
- K-Means Clustering
- DBSCAN Clustering
- Cluster Evaluation
- Business Interpretation

### Day 4
- Time Series Data Preparation
- Stationarity Testing
- Time Series Decomposition

### Day 5
- Baseline Demand Forecasting
- Prophet Model Implementation

### Day 6
- LSTM Model using PyTorch Lightning

### Day 7
- Week 1 Checkpoint
- EDA Report
- Cleaned Dataset
- MLflow Experiment Tracking

## Learning Outcome

By the end of Week 1, I gained practical experience in exploratory data analysis, preprocessing, customer segmentation, time-series forecasting, and experiment tracking. The prepared dataset and baseline models established a solid foundation for the advanced modeling tasks in Week 2.
