# Day 1 – Dataset Selection & Exploratory Data Analysis (EDA)

## Objective

The objective of Day 1 was to select an appropriate dataset for analysis and perform an initial Exploratory Data Analysis (EDA). This step helped in understanding the dataset's structure, identifying data quality issues, and discovering patterns before moving to data preprocessing and model development.

---

## Tasks Performed

### 1. Dataset Selection

Selected a retail sales dataset containing customer transactions, product details, inventory information, and sales records. This dataset was chosen because it is suitable for customer segmentation, demand forecasting, and business analytics.

### 2. Data Exploration

Performed an initial exploration of the dataset to understand its overall structure and contents.

The following checks were carried out:

- Loaded the dataset into Jupyter Notebook.
- Examined the number of rows and columns.
- Viewed the first and last few records.
- Checked column names and data types.
- Generated summary statistics for numerical features.

### 3. Missing Value Analysis

Identified missing values across different columns to determine which features required cleaning or preprocessing.

### 4. Distribution Analysis

Analyzed the distribution of important numerical variables such as Quantity and Price to understand customer purchasing patterns and detect any unusual values.

### 5. Correlation Analysis

Created a correlation heatmap to study the relationships between numerical features and identify variables that may influence future machine learning models.

## Outcome

By completing Day 1, I gained a clear understanding of the dataset, identified missing values and data quality issues, explored feature distributions, and analyzed relationships between variables. This analysis provided the foundation for data cleaning and feature engineering in the next stage of the internship.
