# Day 5 – Baseline Demand Forecasting with Prophet

## Objective

The objective of Day 5 was to build a baseline demand forecasting model using Facebook Prophet. This model helps predict future sales trends by analyzing historical sales data while considering trend and seasonality patterns.

---

## Tasks Performed

### 1. Data Preparation

Prepared the time series dataset for forecasting.

The following tasks were completed:

- Selected historical sales data.
- Formatted the dataset according to Prophet requirements.
- Renamed the date and target columns to `ds` and `y`.
- Checked the dataset for missing values and inconsistencies.

### 2. Prophet Model Development

Built a baseline forecasting model using the Prophet library.

The following steps were performed:

- Initialized the Prophet model.
- Trained the model using historical sales data.
- Generated future dates for prediction.
- Forecasted future sales values.

### 3. Forecast Visualization

Visualized the forecasting results to understand future sales trends.

The following visualizations were created:

- Actual vs Forecasted Sales
- Trend Component
- Seasonal Component
- Forecast Confidence Intervals

### 4. Model Evaluation

Reviewed the forecasting results and analyzed the model's ability to capture sales trends and seasonal patterns.

---

## Tools & Libraries Used

- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Prophet

---

## Outcome

By the end of Day 5, a baseline demand forecasting model was successfully developed using Prophet. The model generated future sales predictions and provided valuable insights into business trends and seasonality, forming the foundation for more advanced forecasting models.
