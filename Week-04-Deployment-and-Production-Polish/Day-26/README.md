# Day 26 – Monitoring Setup with Prometheus and Grafana

## Objective

The objective of Day 26 was to implement application monitoring using Prometheus and Grafana to track system performance and resource utilization.

---

## Tasks Performed

### 1. Monitoring Configuration

Configured monitoring tools.

The following tasks were completed:

- Installed Prometheus.
- Configured Grafana.
- Added monitoring dashboards.
- Connected application metrics.

### 2. Performance Monitoring

Validated monitoring system.

The following tasks were completed:

- Monitored CPU usage.
- Monitored memory usage.
- Verified dashboard metrics.
- Tested alert configuration.

## Outcome

By the end of Day 26, a complete monitoring system was successfully implemented for tracking application performance.
