# Day 22 – Docker Multi-Stage Builds for the Application

## Objective

The objective of Day 22 was to containerize the application using Docker multi-stage builds, creating lightweight and production-ready images for deployment.

---

## Tasks Performed

### 1. Docker Configuration

Configured Docker for the application.

The following tasks were completed:

- Created Dockerfile.
- Implemented multi-stage builds.
- Reduced image size.
- Verified container execution.

### 2. Container Testing

Validated Docker containers.

The following tasks were completed:

- Built Docker image.
- Ran application inside container.
- Tested application functionality.
- Fixed configuration issues.

By the end of Day 22, the application was successfully containerized using Docker multi-stage builds, making it lightweight and deployment-ready.
