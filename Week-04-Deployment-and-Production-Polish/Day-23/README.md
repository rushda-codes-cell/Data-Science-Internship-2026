# Day 23 – Kubernetes Manifests and Deployment Configuration

## Objective

The objective of Day 23 was to prepare Kubernetes manifests and deployment configurations for managing the application in a container orchestration environment.

---

## Tasks Performed

### 1. Kubernetes Configuration

Prepared deployment manifests.

The following tasks were completed:

- Created deployment.yaml.
- Configured service.yaml.
- Defined resource settings.
- Configured replicas.

### 2. Deployment Validation

Verified Kubernetes configuration.

The following tasks were completed:

- Validated YAML files.
- Checked deployment settings.
- Tested service configuration.
- Reviewed deployment workflow.
