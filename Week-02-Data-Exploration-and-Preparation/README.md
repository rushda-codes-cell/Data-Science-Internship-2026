#Week 2 – Advanced Modeling & Churn Prediction

##Overview

Week 2 focused on building advanced machine learning models for demand forecasting and customer churn prediction. During this week, I developed forecasting models, implemented inventory optimization, tuned model performance, monitored data drift, and automated the retraining pipeline.

###Topics Covered

Hybrid Forecasting (Prophet + LSTM)
Churn Prediction using XGBoost
Inventory Optimization
Feature Importance Analysis
Hyperparameter Tuning (Optuna)
Data Drift Detection
Automated Retraining Pipeline (Airflow)

###Outcome

By the end of Week 2, advanced forecasting and churn prediction models were developed, inventory optimization logic was implemented, model performance was improved through hyperparameter tuning, data drift monitoring was established, and an automated retraining pipeline was created.
