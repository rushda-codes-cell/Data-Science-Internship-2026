# Day 9 – Customer Churn Prediction using XGBoost

## Objective

The objective of Day 9 was to develop a customer churn prediction model using XGBoost and understand the factors influencing customer churn through model explainability.

---

## Tasks Performed

### 1. Data Preparation

Prepared the customer dataset for churn prediction.

The following tasks were completed:

- Selected relevant customer features.
- Split the dataset into training and testing sets.
- Handled missing values and encoded categorical variables.

### 2. XGBoost Model

Built a machine learning model for customer churn prediction.

The following tasks were completed:

- Trained the XGBoost classifier.
- Evaluated model accuracy.
- Generated classification metrics.
- Compared predicted and actual results.

### 3. Model Explainability

Interpreted model predictions using SHAP.

The following tasks were completed:

- Generated SHAP values.
- Identified important features.
- Explained customer churn predictions.

---

## Tools & Libraries Used

- Python
- Jupyter Notebook
- Pandas
- XGBoost
- SHAP
- Scikit-learn

---

## Outcome

By the end of Day 9, an accurate customer churn prediction model was developed and key factors influencing customer churn were successfully identified using SHAP explainability.
