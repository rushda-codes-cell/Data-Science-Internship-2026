# Day 8 – Hybrid Forecasting Model (Prophet + LSTM)

## Objective

The objective of Day 8 was to improve demand forecasting accuracy by developing a hybrid forecasting model that combines Prophet and LSTM. This approach leverages Prophet for capturing trend and seasonality while using LSTM to learn complex temporal patterns in the sales data.

---

## Tasks Performed

### 1. Prophet Forecasting

Built a forecasting model using Prophet to capture trends and seasonality in the sales data.

The following tasks were completed:

- Prepared time-series data.
- Built the Prophet forecasting model.
- Generated future demand predictions.
- Evaluated forecasting performance.

### 2. LSTM Forecasting

Developed an LSTM model to learn complex temporal dependencies in historical sales data.

The following tasks were completed:

- Prepared sequential input data.
- Built and trained the LSTM model.
- Generated demand forecasts.
- Compared predicted values with actual sales.

### 3. Hybrid Ensemble Model

Combined Prophet and LSTM predictions to improve forecasting accuracy.

The following tasks were completed:

- Combined predictions from both models.
- Compared forecasting performance.
- Evaluated the hybrid model.

## Outcome

By the end of Day 8, a hybrid forecasting model was successfully developed by combining Prophet and LSTM. The ensemble approach improved forecasting performance and established a stronger foundation for advanced demand prediction.
