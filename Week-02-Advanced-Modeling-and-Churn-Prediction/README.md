# Week 2 – Advanced Modeling & Churn Prediction

## Overview

The second week of the internship focused on developing advanced machine learning models for demand forecasting and customer churn prediction. During this week, I built hybrid forecasting models, implemented churn prediction using XGBoost, optimized inventory planning, improved model performance through hyperparameter tuning, monitored data drift, and automated the model retraining pipeline.

The primary objective was to enhance prediction accuracy, improve model interpretability, establish monitoring mechanisms, and automate the machine learning workflow for production-ready deployment.
