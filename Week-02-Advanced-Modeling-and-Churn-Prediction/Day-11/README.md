# Day 11 – Feature Importance & Hyperparameter Tuning

## Objective

The objective of Day 11 was to improve machine learning model performance by analyzing feature importance and optimizing model hyperparameters using Optuna. These techniques help identify the most influential features and achieve better predictive accuracy.

---

## Tasks Performed

### 1. Feature Importance Analysis

Analyzed the contribution of each feature to the machine learning model.

The following tasks were completed:

- Identified important features.
- Ranked features based on importance.
- Visualized feature importance.
- Removed less significant features.

### 2. Hyperparameter Tuning

Optimized the machine learning model using Optuna.

The following tasks were completed:

- Defined the optimization search space.
- Performed automated hyperparameter tuning.
- Compared model performance.
- Selected the best-performing model.

---

## Tools & Libraries Used

- Python
- Jupyter Notebook
- Pandas
- Scikit-learn
- Optuna
- Matplotlib

---

## Outcome

By the end of Day 11, the model achieved improved performance through feature importance analysis and optimized hyperparameters, resulting in better prediction accuracy.
