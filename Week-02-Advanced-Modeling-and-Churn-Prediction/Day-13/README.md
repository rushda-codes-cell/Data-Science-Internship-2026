# Day 13 – Automated Retraining Pipeline

## Objective

The objective of Day 13 was to automate the machine learning retraining workflow using Apache Airflow. Automation ensures that models remain updated as new data becomes available.

---

## Tasks Performed

### 1. Airflow Workflow

Designed an automated retraining pipeline.

The following tasks were completed:

- Created an Airflow DAG.
- Scheduled pipeline execution.
- Automated data preprocessing.
- Configured model retraining.

### 2. Pipeline Automation

Validated the automated workflow.

The following tasks were completed:

- Tested pipeline execution.
- Verified scheduled runs.
- Checked workflow dependencies.
- Monitored pipeline status.

---

## Tools & Libraries Used

- Python
- Apache Airflow
- Pandas

---

## Outcome

By the end of Day 13, an automated retraining pipeline was successfully developed, enabling continuous model updates with minimal manual intervention.
