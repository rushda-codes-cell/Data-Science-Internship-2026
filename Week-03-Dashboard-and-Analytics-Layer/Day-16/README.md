# Day 16 – Demand Forecasting Dashboard & What-If Analysis

## Objective

The objective of Day 16 was to develop an interactive demand forecasting dashboard and implement what-if analysis for evaluating different sales scenarios.

---

## Tasks Performed

### 1. Forecast Dashboard

Developed forecasting visualizations.

The following tasks were completed:

- Displayed monthly sales trends.
- Added forecast charts.
- Visualized prediction results.
- Compared historical and forecasted values.

### 2. What-If Analysis

Implemented scenario analysis.

The following tasks were completed:

- Added user input controls.
- Simulated demand variations.
- Updated forecasts dynamically.
- Validated dashboard outputs.

---

## Tools & Libraries Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Scikit-learn

---

## Outcome

By the end of Day 16, an interactive demand forecasting dashboard with what-if analysis was successfully implemented for better business decision-making.
