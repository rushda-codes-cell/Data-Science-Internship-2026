# Day 17 – Customer Churn Analytics Dashboard

## Objective

The objective of Day 17 was to build an interactive dashboard for customer churn analysis, enabling users to monitor churn patterns and key business metrics.

---

## Tasks Performed

### 1. Churn Dashboard

Developed customer analytics.

- Displayed churn statistics.
- Added customer segmentation.
- Visualized churn distribution.
- Created KPI cards.

### 2. Dashboard Testing

Validated dashboard functionality.

The following tasks were completed:

- Tested filters.
- Verified charts.
- Checked dashboard responsiveness.
- Improved usability.

## Outcome

By the end of Day 17, an interactive customer churn analytics dashboard was successfully developed for monitoring customer behavior and churn trends.
